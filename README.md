# 🌦️ Weather App

A modern and responsive weather application that provides real-time weather information for any city around the world. Users can search for a location and instantly get current weather details including temperature, humidity, wind speed, and weather conditions.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature updates
- 💧 Humidity information
- 🌬️ Wind speed details
- ☁️ Weather condition display
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- OpenWeather API (or your weather API)
